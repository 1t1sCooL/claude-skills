# claude-skills

Личная коллекция скиллов и агентов для [Claude Code](https://claude.com/claude-code).
Скиллы лежат в `skills/`, агенты — в `agents/`. Устанавливаются копированием в
`~/.claude/` (см. [AGENTS.md](AGENTS.md) — инструкция для AI-агента, который ставит всё сам).

> Личные пути в скиллах обобщены до `$HOME/...` и плейсхолдеров (`your-site.com`,
> `your-project`), чтобы коллекция была переносимой. Подставь свои значения после установки.

| **migrate** | `/migrate` | Зеркалирует текущий git-проект в личный GitHub с переписыванием авторов коммитов. Требует `~/.claude/secrets.env` (см. AGENTS.md). |
| **project-backlog** | `/project-backlog` | Генерирует список актуальных задач проекта в формате «дата + bullets» (RU). |
| **daily-standup** | — | Дневной отчёт о работе в кратком деловом стиле: собирает сделанное из git-истории дня и сессии (RU). |

## Слой `aif-*` (надстройка над AI Factory)

24 скилла `aif-*` — личная надстройка над [AI Factory](https://skills.sh). Покрывают полный
цикл: `/aif` (сетап), `/aif-plan` → `/aif-implement` → `/aif-verify`, ревью/безопасность/доки,
докеризация, CI, build-automation и др. Полный список — в `skills/aif*`.

## Установка

Быстро — отдай [AGENTS.md](AGENTS.md) любому AI-агенту Claude Code, он установит всё сам.
Вручную:

```bash
git clone https://github.com/1t1sCooL/claude-skills.git
cd claude-skills
./install.sh        # копирует skills/ и agents/ в ~/.claude/, регистрирует триггеры
```

Личные скиллы — делай что хочешь. Слой `aif-*` основан на AI Factory (см. их upstream-лицензию).
