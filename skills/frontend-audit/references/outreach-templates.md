# Шаблоны: мини-аудит + cold outreach

> Цифры — публичный Lighthouse. Подавать конструктивно. Персонализировать под конкретный сайт.

## Формат мини-аудита (на сайт)

```markdown
## <Название> (<домен>) — <ниша>

| Метрика | Mobile | Desktop |
|---|---|---|
| **Performance** | 🔴/🟠/🟢 <N> | <N> |
| LCP | <s> | <s> |
| TBT | <ms> | <ms> |
| CLS | <v> | <v> |
| Speed Index | <s> | <s> |
| Security-заголовки | <X из 6> | |

**Что не так:** 2–3 пункта с цифрами и человеческим объяснением последствий
(«TBT 1.2 с → кнопки не реагируют секунду на старте»).

**Что починить (первым):** 1–2 строки, реалистичный таргет.

**Вывод одной строкой:** *<цепкая формулировка проблемы>.*
```

Эмодзи-пороги Performance: 🔴 0–49 · 🟠 50–89 · 🟢 90–100.

## Шаблон cold-outreach (персональный, value-first)

Подставлять реальные цифры сайта. Коротко, без воды, без давления.

**Короткий заход (Telegram/DM/email, первое касание):**
```
Здравствуйте! Прогнал <домен> через Lighthouse — на мобильных Performance
<N>/100, <главная цифра: напр. LCP <Y> с / Speed Index <Z> с>. Нашёл 3 вещи,
которые тормозят загрузку и бьют по конверсии (и по позициям в поиске —
Google учитывает скорость).

Если интересно — пришлю короткий разбор бесплатно, без обязательств.
```

**Если ответили — разбор (второе касание):**
```
Вот 3 главные:
• <проблема 1 с цифрой> — <последствие>
• <проблема 2 с цифрой> — <последствие>
• <проблема 3 с цифрой> — <последствие>

Это чинится за 1–3 недели и обычно даёт <реалистичный эффект>. Делаю
продуктизированные аудиты с фикс-ценой и фикс-объёмом — детали и пакеты тут:
mmalabugin.ru/audit. Могу сделать полный разбор по вашему сайту.
```

### Правила переписки
- Первое касание — **дать пользу**, не продавать. Предложить бесплатный короткий разбор.
- Один сайт = одно персональное письмо. Цифры реальные и свежие.
- Не более 2 касаний без ответа. Не давить.
- Тон спокойный, экспертный. Никакого «у вас всё плохо».

## Пакеты услуги (для ответов на «сколько стоит»)
- **Express** — от 25 000 ₽, 3–5 дней: 1 страница, авто-замеры, отчёт top-10 + чек-лист.
- **Standard** — от 60 000 ₽, 1–1.5 нед: до 5 страниц, ручной разбор, приоритизация, Loom.
- **Pro/Deep** — от 120 000 ₽, 2–3 нед: глубокий аудит, бандл, diff-рекомендации, созвон + follow-up.
- **Re-audit** — от 15 000 ₽: подтверждение «было → стало» (используй `--compare`).
